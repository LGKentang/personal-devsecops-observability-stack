# Personal DevSecOps Observability Stack

A cloud-native learning platform demonstrating:

- Go microservices
- Kubernetes deployments
- Terraform IaC provisioning (Grafana Tempo, Loki, Mimir, Prometheus)
- Full OpenTelemetry tracing pipeline
- GitHub Actions CI/CD with GHCR
- DevSecOps security tooling


Designed as a portfolio system, not just an application.
